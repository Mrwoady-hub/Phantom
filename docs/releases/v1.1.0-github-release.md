# Phantom v1.1.0

![Phantom v1.1.0 release banner](../assets/release-banner-v1.1.0.svg)

Phantom v1.1.0 is the transition release that aligns the project around its current product identity and public repository presence.

## Highlights

- Renamed the repository and project identity to `Phantom`
- Added a stronger GitHub landing page with screenshots, architecture media, and demo assets
- Improved repository hygiene by removing local workspace artifacts and Xcode user-state from source control
- Added release notes, contribution guidance, and a security policy

## Included in This Release

- new `Phantom` README and public-facing project positioning
- release media:
  - dashboard screenshot
  - capabilities strip
  - architecture overview
  - scan-flow GIF
  - dark `v1.1.0` release banner
- repository policy files:
  - `LICENSE`
  - `CONTRIBUTING.md`
  - `SECURITY.md`

## Notes

- Existing clones should use the renamed GitHub repository path:
  - `git@github.com:Mrwoady-hub/Phantom.git`
- Local Xcode `xcuserdata` is no longer intended to be committed.
